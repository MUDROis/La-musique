## 📖 Описание проекта

**La Musique** — это обучающая игра, направленная на развитие лексических навыков, аудирования и говорения по французскому языку. Приложение состоит из четырёх интерактивных модулей, каждый из которых способствует усвоению новой лексики и отработке языковых структур в игровой форме.

### Основные темы:
- Названия музыкальных жанров (rock, jazz, classical, electronic)
- Музыкальные термины (chanson, chanteur, instrument, guitare)
- Выражение предпочтений (j’aime, je n’aime pas, j’adore, je déteste)
- Глаголы (écouter, jouer)
- Практика диалогов по теме «Музыка»

## 🏗️ Структура приложения

Приложение включает 4 интерактивных модуля:

| Модуль | Название | Описание |
|--------|----------|----------|
| 🎧 | **Warm-up** | Аудирование: слушайте аудиофрагменты и определяйте музыкальный жанр |
| 📖 | **Flashcards** | Карточки: изучайте и запоминайте лексику (французское слово + английский перевод) |
| 🧠 | **Quiz** | Викторина: проверьте знание переводов французских слов |
| 💬 | **Dialogue** | Диалоги: читайте и слушайте реальные разговорные ситуации |

## 🎓 Методические рекомендации

### Для преподавателей

#### 📋 Рекомендации по использованию в классе

1. **Предварительная подготовка**
   - Объясните базовую лексику перед использованием игры.
   - Проведите краткий разговор о музыкальных жанрах на родном языке.
   - Задайте установку: *«Сегодня мы изучим, как говорить о музыке по-французски»*.

2. **Warm-up (Разминка)**
   - **Цель**: активизировать аудирование, познакомить с звучанием жанров.
   - **Как проводить**:
     - Демонстрируйте первый жанр, слушайте вместе.
     - Попросите учеников высказать предположения: *«Как вы думаете, какой это жанр?»*
     - По завершении — обсуждение: *«Что вам напомнило это звучание?»*
   - **Важно**: не стесняйтесь ошибок — это этап включения в тему.

3. **Flashcards (Карточки)**
   - **Цель**: ввести новую лексику в запоминание.
   - **Как проводить**:
     - Покажите карточку, произнесите слово вслух (кнопка 🔊).
     - Попросите учеников повторить за приложением.
     - Используйте карточки для быстрой проверки: *«Кто может перевести это слово?»*
     - Дополнительно: ученики могут сделать карточки на бумаге (рисунок + французское слово).

4. **Quiz (Викторина)**
   - **Цель**: закрепить лексику в активной памяти.
   - **Как проводить**:
     - Инструкция: *«Выберите правильный перевод»*.
     - После ответа обсуждение: *«Почему это верно?»* / *«Что помогло вам выбрать этот вариант?»*
     - Для слабых учеников: разрешите использовать карточки как справочный материал.

5. **Dialogue (Диалоги)**
   - **Цель**: показать лексику в контексте реальной речи.
   - **Как проводить**:
     - Сначала прослушайте диалог целиком.
     - Затем разберите построчно: читаем, переводим, повторяем за приложением.
     - **Ключевая задача**: выделить ключевые слова и выражения (жанры и предпочтения).
     - Дополнительное задание: попросите учеников составить свой диалог по аналогии.

#### 🎯 Методические цели

| Модуль | Развиваемые навыки | Языковые цели |
|--------|-------------------|---------------|
| Warm-up | Аудирование, прогнозирование | Распознавание жанров по звучанию |
| Flashcards | Восприятие на слух, произношение | Запоминание лексики |
| Quiz | Чтение, выбор правильного варианта | Активизация лексики |
| Dialogue | Чтение, аудирование, говорение | Использование лексики в речи |

#### 💡 Советы

- Игра подходит как для индивидуальной, так и для парной/групповой работы.
- Используйте её в начале урока (разминка), во время (отработка темы) или в конце (проверка и закрепление).
- Для домашней работы: назначьте пройти игру целиком как подготовку к следующему уроку.
- Поощряйте повторное прохождение — запоминание происходит при многократном контакте.

### Для родителей

#### 📌 Что делает ваш ребёнок?

Ваш ребёнок учит французский язык и тренирует лексику по теме «Музыка». Игра помогает:
- Запомнить новые слова (названия жанров, музыкальные термины, выражения предпочтений)
- Научиться понимать французскую речь на слух
- Развить навык говорения (повторение за приложением)
- Применять лексику в диалогах

#### 🎮 Как помогать ребёнку дома?

1. **Создайте комфортную обстановку**
   - Выделите 15–20 минут без отвлечений.
   - Рядом может быть кто-то, кто знает французский (или хотя бы английский).
   - Убедитесь, что у вас есть колонки или наушники (для аудирования).

2. **Советы по прохождению**
   - **Warm-up**: Сначала пусть ребёнок просто слушает, потом пробует угадать. Не критикуйте ошибки — это тренировка уха.
   - **Flashcards**: Поощряйте повторение слов вслух. Можно вместе с вами (вы повторяете за приложением, ребёнок — за вами).
   - **Quiz**: Если ребёнок ошибся, вместе разберите правильный ответ. Обсудите: *«Как ты думаешь, почему это верно?»*
   - **Dialogue**: Прочитайте диалог вместе, посмотрите перевод. Попробуйте сыграть его вдвоём (вы — один персонаж, ребёнок — другой).

3. **Как мотивировать**
   - Хвалите за прогресс, даже небольшой.
   - Отмечайте, что ребёнок уже запомнил.
   - Пусть он покажет вам, как работает игра — это усилит его чувство собственного достоинства.

#### 📊 Как оценивать результат?

- Оценка в игре — не конечная цель, а показатель прогресса.
- Главное — понимание и готовность говорить.
- Если ребёнок не справляется с каким-то модулем — вернитесь к нему позже, после повторения.

## ⚙️ Техническая информация

### Технологии
- **HTML5** — структура приложения
- **CSS3** — стилизация и анимации
- **JavaScript (ES6+)** — логика игры
- **Web Audio API** — синтез аудио (для жанров без звуковых файлов)
- **Web Speech API** — синтез речи (озвучивание слов)

### Поддерживаемые браузеры
- Google Chrome (рекомендуется)
- Mozilla Firefox
- Microsoft Edge
- Safari

### Системные требования
- Операционная система: Windows / macOS / Linux / iOS / Android
- Браузер с поддержкой HTML5 Audio и Speech Synthesis
- Подключение к интернету (не обязательно — игра работает локально)

## 🚀 Как запустить

### Вариант 1: Локальный запуск (без сервера)
1. Скачайте все файлы в одну папку.
2. Откройте `index.html` в любом современном браузере.
3. Готово!

### Вариант 2: Через локальный сервер (для полной работы с аудиофайлами)
```bash
# В папке проекта выполните:
npx serve .
# или
python -m http.server 8000
```
Затем откройте `http://localhost:8000` в браузере.
